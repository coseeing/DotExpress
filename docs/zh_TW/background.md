# 背景

在視覺障礙者的閱讀與學習世界中，「點字」始終是不可或缺的核心媒介。然而，長期以來，點字轉譯工作受限於工具不足、系統老舊，尤其在中文文本夾雜英文、符號或專有名詞時，往往必須仰賴人工反覆判斷與修正，不僅耗時，也容易產生歧義。

DotExpress 正是在這樣的需求背景下誕生。DotExpress 是一套專為實務應用而設計的「文字轉點字」轉譯軟體，可將一般中文、英文及多語文本，轉換為固定行寬、可直接印製的點字版面，作為紙本點字教材與文件製作的基礎來源資料。其核心特色在於：能正確處理中文文本中夾雜的英文內容，並可依需求選擇 UEB 一級或 UEB 二級點字規則，大幅提升點字閱讀的一致性與專業性。

對於視障點譯人員而言，DotExpress 不只是一套新工具，更是一個時代的轉換——點譯工作終於不必再被侷限於舊有系統，視障點譯員可以真正走進 Windows 圖形化環境，自主、自由且高效率地完成專業點字轉譯工作。

對於特殊教育教師、點字轉譯人員，DotExpress 能有效減輕教材製作的負擔，提升轉譯效率與精確度；對於家長與教學助理，也能即時將課堂教材、講義或補充資料轉換為點字，讓視障孩子不再因等待而落後學習進度。

更重要的是，DotExpress 從設計之初即以「明盲共用、無障礙操作」為核心理念。無論是視障使用者或明眼使用者，都能在同一套軟體中順利操作、共同協作，真正實現融合的教育與工作環境。

DotExpress 不只是文字轉點字的工具，它是一座橋梁——連結教材與閱讀、專業與效率、也連結視障者在數位時代中，能夠自主進行點譯工作，並安心發揮專業的可能。
