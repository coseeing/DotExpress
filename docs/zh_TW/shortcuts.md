# 快捷操作列表

下表整理 DotExpress 目前與編輯區、文件列表相關的快捷操作與常用互動。

| 區域 | 操作 | 快捷鍵 / 滑鼠 | 說明 |
| --- | --- | --- | --- |
| `來源文字編輯區` | 轉換 | `Ctrl+Enter` | 執行轉換，效果等同按下「轉換」按鈕。 |
| `來源文字編輯區`/`點字編輯區` | 放大字型 | `Ctrl + 滑鼠滾輪向上` | 將編輯區字型大小增加。 |
| `來源文字編輯區`/`點字編輯區` | 縮小字型 | `Ctrl + 滑鼠滾輪向下` | 將編輯區字型大小減少。 |
| `點字編輯區` | 匯出 `.brl` | `Ctrl+S` | 匯出目前開啟文件的點字內容為 `.brl` 檔。 |
| 文件列表 | 開啟文件 | `Enter` | 開啟目前選取的文件，並將內容載入右側編輯區，效果等同右鍵選單中的「開啟」。 |
| 文件列表 | 重新命名 | `F2` | 對目前選取的文件執行重新命名，效果等同右鍵選單中的「重新命名」。 |
| 文件列表 | 開啟右鍵選單 | 滑鼠右鍵 | 開啟文件列表的操作選單。 |

## 文件列表右鍵選單

| 項目 | 說明 |
| --- | --- |
| `Open` | 開啟目前選取的文件。 |
| `Delete` | 刪除目前選取的文件。 |
| `Delete All` | 刪除全部文件；確認後會清空目前工作區文件，並立即要求建立新的第一份文件。 |
| `Add` | 新增一份文件。 |
| `Rename` | 重新命名目前選取的文件。 |
| `Import` | 匯入單一文件。 |
| `Export` | 匯出單一文件。 |
| `Batch Import` | 批次匯入多份文件。 |
| `Batch Export` | 批次匯出多份文件。 |

## 備註

* 字型大小同步 | 使用 `Ctrl + 滑鼠滾輪` 調整字型時，`來源文字編輯區`、`點字編輯區` 與 `View > Font Size` 會保持同步。
* 一般滾輪行為 | 未按住 `Ctrl` 時，滑鼠滾輪仍維持原本的捲動行為。
